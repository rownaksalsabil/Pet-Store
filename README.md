Pet Store E-Commerce Website
A full-stack e-commerce web application designed for buying and selling pets, pet accessories, and nutrition products.
 Built as part of an academic design project.
This platform allows:
Users to browse and search pet-related products
Registered users to add items to cart and place orders
Admin to manage products and process orders
The system is designed to be secure, responsive, and user-friendly, targeting pet owners and pet lovers.
